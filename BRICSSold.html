bri{css}

Both a digital and physical desktop toolkit designed to teach web development bri{css} is a project developed during Carnegie Mellon's 2018 TartanHacks

Coded in Java, bri{css} is a toolkit intended to educate and introduce children to coding concepts in a more tangible way. As the team came together, we were all, on some level, trying to code our own websites. All with some experience coding, we found it difficult to simply just jump in head first and begin. Thus we created bri{css}, a toolkit that combines physical understanding with a digital interface to help visually introduce basic web development.

As project manager, I was responsible for the proof of concept, hardware, and the physical toolkit. In addition, I collaborated on the visuals for the graphic user interface design.
