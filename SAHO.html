Potential Home Screen Digital Wireframes






I explored several different themes. Frankly, I preferred the black and white icons with the colored overlay, however it was hard to discern what the images were. Taking into account how users interact with digital thresholds, I didn't want to create a pointless distracting element. Thus I opted for the colorful feature thumbnails.







Over the course of this project, the site underwent several design changes







After choosing to go with the colorful thumbnails, I created simplified coverpage icons for each page to remain cohesive. While doing this, I developed a style for the elements on the website and became rather ambitious, wanting to create more interactive features.

Up until this point, I had become the most familiar with html; however with myy decision to add more interactive elements to the site, I thrust myself into scss and js. Using online resources, I was able to produce a version of the GUI in one of my most recent projects, bri{css}.
